**AUDIT FOCUS: Runtime Safety - Exotic Types (tryte, nyte, etc.)**

**Threat Model:**
Exotic types (base-3 tryte, base-9 nyte) are used for certain neural network models and ternary logic. While less common, they interface with safety-critical systems. A bug in tryte→binary conversion could cause: ternary decision (yes/no/unknown) → binary (yes/no) loses "unknown" → robot acts on uncertain data → wrong decision → harm.
