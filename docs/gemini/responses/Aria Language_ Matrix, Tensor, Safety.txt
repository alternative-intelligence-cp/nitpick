2.1 The Two’s Complement Singularity and the TBB Correction
The necessity for "Twisted" types stems from the inherent flaw in standard Two's Complement arithmetic, which dominates modern CPU architecture. In an $N$-bit signed integer, the representable range is $[-2^{N-1}, 2^{N-1}-1]$. For an 8-bit integer, this is $[-128, +127]$.
The value $-128$ (0x80) is a "singularity." It has no positive counterpart (since $+128$ cannot be represented). This asymmetry breaks the fundamental symmetry required for wave physics. In a wave processor, phase cancellation relies on the axiom that for every amplitude $A$, there exists an inverse $-A$ such that $A + (-A) = 0$. If the system attempts to invert a wave with amplitude $-128$, standard arithmetic either overflows or returns the identity, introducing a Direct Current (DC) bias into the interference pattern. Over billions of iterations in the AGI's self-improvement loop, this bias accumulates, destabilizing the toroidal field.1
Aria's Twisted Balanced Binary (TBB) types (tbb8, tbb16, tbb32, tbb64) resolve this by artificially restricting the domain to be perfectly symmetric and reserving the vestigial bit pattern as an error sentinel.
* Valid Domain: $[-127, +127]$ (for 8-bit).
* Sentinel: -128 (0x80) is ERR.
2.2 The Algebra of Sentinel Propagation
We define the algebraic structure of TBB as a commutative ring $R_{TBB}$ with an absorbing element $\bot$ (ERR). The operations $\oplus$ (addition) and $\otimes$ (multiplication) are defined such that:
1. Absorption: $\forall x \in R_{TBB}, x \oplus \bot = \bot$ and $x \otimes \bot = \bot$.
2. Symmetry: $\forall x \neq \bot, \exists (-x) \in R_{TBB}$ such that $x \oplus (-x) = 0$.
3. Closure: If an operation on valid elements exceeds the domain (overflow), the result is $\bot$.
This "Sticky Error" logic is the mathematical enforcement of the user's safety requirement. If a single neuron in the AGI's Mamba layer calculates an invalid weight or encounters a sensor glitch, the ERR value replaces the result. In standard systems, this might result in a NaN that propagates slowly or is silently ignored. In TBB, the ERR state propagates at the speed of the SIMD pipeline, effectively "turning off" that branch of calculation instantly.

Standard CPU instructions do not respect the Aria ERR sentinel. 127 + 1 in standard int8 wraps to -128. In Aria tbb8, 127 + 1 must result in ERR (-128), and ERR + 1 must remain ERR.

Algorithm: Safe Vector Addition (vec9_add)
1. Load: Load Vector A and Vector B into registers.
2. Sentinel Mask Generation:
   * MaskA = (A == ERR)
   * MaskB = (B == ERR)
   * MaskAnyErr = MaskA | MaskB
3. Arithmetic & Overflow Check:
   * Perform standard addition using LLVM intrinsics (llvm.sadd.with.overflow).
   * This returns a struct {Result, OverflowBit}.

4. Failure Union:

5. Sentinel Blending:
   * Use a vector select (blend) instruction.
   * FinalResult = Select(MaskFail, ERR_VECTOR, RawResult)
